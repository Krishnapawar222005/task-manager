## Features

* User Registration
* User Login
* JWT Authentication
* Create Tasks
* View Tasks
* Delete Tasks
* MongoDB Database
* React Frontend
* Express Backend
* Responsive UI

## Tech Stack

### Frontend

* React
* Axios
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

## Installation

### Backend

cd server
npm install
node server.js

### Frontend

cd client
npm install
npm run dev
